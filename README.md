# 🍽️ Safe to Eat

Dashboard interativo com mapa de restaurantes seguros para celíacos em Curitiba e Região Metroplitana.

## 💡 Sobre o projeto

O **_Safe to Eat_** nasceu como um projeto de portfólio com foco em análise de dados, visualização geográfica e desenvolvimento de dashboards interativos. 

A ideia surgiu da dificuldade de pessoas celíacas enfrentam ao tentar encontrar locais realmente seguros para alimentação fora de casa. 

O projeto utiliza dados reais coletados manualmente e transformados em uma aplicação visual interativa utilizando **_Python_**. 

Além da parte técnica, o projeto também busca gerar **_impacto social positivo_**, centralizando informações úteis para a comunidade celíaca. 

## ⚙️ Tecnologias utilizadas

- Python
- Pandas
- Streamlit 
- Folium 
- Git e GitHub

## ✅ Etapas já realizadas

- **Coleta de dados:** coletados manualmente a partir de informações disponíveis no Google Maps. 

- **Limpeza e tratamento de dados:** 
    - Padronização de colunas;
    - Correção de espaços extras; 
    - Tratamento de valores nulos; 
    - Ajustes de nomenclatura;
    - Organização de categorias.

- **Desenvolvimento do Dashboard:** mapa intereativo criado utilizando Streamlit para a interface e Folium para visualização geográfica. Cada restaurante aparece como um marcador no mapa.
